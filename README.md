# web-app-test-automation-tool

## 概要
非エンジニア向けのテスト自動化ツール

### 使用ツール
- 入出力：スプレッドシート
- テスト実行：Selenium IDE

### 使用の流れ
1. スプレッドシートにてテストケースを作成
2. テスト実行ファイルをローカルに保存
3. Selenium IDEにて実行ファイルを読み込み、テストを実施
4. テスト結果をスプレッドシートに出力
